Séance 1

Introduction

Les langages HTML et CSS sont les principaux langages du Web.

L'importance des deux langages est schématisée dans les deux photos précédentes :

  • Le HTMLest utilisé pour construire la structure (le squelette) d'une page Web.
  • Le CSS est utilisé pour embellir et donner une apparence attractive de la page Web.

Les deux langages sont complémentaires.

Structure d'une page Web

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  
</body>
</html>

Explication

<!DOCTYPE html>

Indique au navigateur que la page utilise la version 5, selon W3C, du langage HTML

<html lang="en">
...
</html>

Indique le début et la fin du document HTML, ainsi que la langue utilisée dans ce document, utile pour les screen readers.

<head>
...
</head>

Marque l'entête du document HTML. C'est la partie invisible de la page Web qui contient des instructions destinées au navigateur pour l'aider à bien afficher la page.

On y trouve par exemple :

<meta charset="UTF-8">

Indique l'encodage des caractères, l'UTF-8 est un encodage universel qui permet d'afficher toutes les langues (Latin, Arabe, Chinois, Japonais, etc.).

<meta name="viewport"
content="width=device-width, initial-scale=1.0">

Indique que la page peut-être rendue sur un mobile sans subir d'agrandissements, ni de rétricissements.

<title>Document</title>

Indique le texte qui s'affiche dans la barre du titre du navigateur.

<body>
  ...
</body>

Tout le contenu qui sera affiché dans le navigateur doit être écrit à l'intérieur de cette balise.

Les premières balises

Nous voulons concevoir la page web suivante :

La page est composée de plusieurs éléments visibles :

  • Une liste non ordonnée
  • Un titre et un sous-titre
  • Des paragraphes
  • Des images

Liste non ordonnée

Pour construire la liste :

On peut écrire le code suivant:

<ul>
  <li>What is it?</li>
  <li>What is used for?</li>
  <li>What types?</li>
  <li>What is composed of?</li>
</ul>

Dans ce code <ul>...</ul> indiquent le début et la fin de la liste non ordonnée (ul=unordered list).

Tandis que <li>...</li> indiquent les éléments de la liste (li = list item).

Les titres

Le langage HTML supporte 6 niveaux pour les titres, le titre principal est indiqué par un titre niveau 1 <h1>.

Les titres montrés dans l'image :

Sont écrites en HTML :

<h1>Computers</h1>
<h2>What is a computer?</h2>

Les paragraphes

La balise <p>...</p> délimite les paragaphes dans un document HTML. Anisi, les paragraphes ci-dessous :

Peuvent être écrites en HTML de la façon suivante :

<p>A computer is a <b>machine</b> that can be instructed to carry out 
  sequences of arithmetic or logical operations automatically via <i>computer 
  programming</i>. Modern computers have the ability to follow generalized sets 
  of operations, called <b>programs</b>. These programs enable computers 
  to perform an extremely wide range of tasks.</p>
<p>A "complete" computer including the hardware, the operating system 
  (main software), and peripheral equipment required and used for "full" 
  operation can be referred to as a <b>computer system</b>. This term may
  as well be used for a group of computers that are connected and work 
  together, in particular a <u>computer network</u> 
  or <u>computer cluster</u>.</p>

Mise en forme

On remarque qu'il y a des mots soulignés, en gras et en italique.

Nous utilisons la balise :

  • <u>souligné</u> pour souligner un texte : souligné.
  • <b>Gras</b> pour mettre un texte en gras : Gras.
  • <i>Italique</i> pour mettre un texte en italique : Italique.

Les images

La balise img est utilisée pour insérer une image.

Exemple :

<img src="images/html_css_logo.png" alt="Logo HTML et CSS">

L'attribut src indique l'emplacement de l'image.

L'attribut alt indique le texte à afficher en cas de problème d'affichage de l'image.

Pour insérer les images de la figure suivante :

On peut écrire le code HTML suivant :

  <img src="images/computer03_thumb.jpg" alt="Macintosh">

Séance 2

Les liens

Définition

Le mot HTML est l'HyperText Markup Language, langage de balisage hypertexte. D'après Wikipedia :

Un hypertexte est un document ou un ensemble de documents contenant des unités d'information liées entre elles par des hyperliens. Ce système permet à l'utilisateur d'aller directement à l'unité qui l'intéresse, à son gré, d'une façon non linéaire.

Source Wikipedia

Types

Il existe trois types de liens :

  • Les liens externes : utilisés pour référencer des ressources dans d'autres sites.
  • Les liens internes : utilisés pour référencer les ressources dans un même site.
  • Les liens nommés (signets) : utilisés pour référencer des parties d'une même page.
<!-- Lien externe -->
<a href="https://www.google.com">Ouvrir google</a>
<!-- Lien intene -->
<a href="page2.html">Ouvrir page 2</a>
<!-- Lien nommé -->
<a href="#section_2">Section 2</a>
<!-- La destination d'un lien nommé -->
<h2 id="section_2">Partie n°2</h2>

Exemples de types de liens

Attributs

La balise <a> comporte trois parties importantes :

  • L'attribut href="https://www.google.com" indique la cible du lien.
  • L'attribut target="fenetre" indique le nom de la fenêtre ou un mot clé (_blank) dans laquelle le lien sera ouvert.
  • Le texte/l'image qui servira de lien cliquable.

Feuilles de style (CSS)

Introduction

Les feuilles de style en cascade sont définis par le site Mozilla Developper Network (MDN). comme suit :

Les Cascading StyleSheets — ou CSS — (Feuilles de style en cascade) sont la première technique à apprendre après le HTML. Alors que HTML s'utilise pour définir la structure et la sémantique du contenu, les CSS sont employées pour composer et déterminer l'apparence de ce contenu.

Ainsi par exemple, vous utiliserez les CSS pour modifier les polices, la couleur, la taille et l'espacement de votre contenu, pour le répartir sur plusieurs colonnes ou bien pour ajouter des animations et autres fonctionnalités décoratives.

Source MDN

Premiers pas

Il existe trois méthodes pour appliquer du CSS à un document HTML. La plus utilisée est celle qui consiste à :

  • Créer un fichier style.css externe,
  • Et le lier au fichier HTML à travers la balise <link>, qui doit figurer dans la partie <head> du document HTML.
Exemple
<link rel="stylesheet" href="style.css">

Les règles CSS

Le code tapé dans le fichier css est appelé une règle. Cette règle est constituée de déclarations qui modifient les valeurs des propriétés du document.

Les sélecteurs

Outre le sélecteur body, nous devons utiliser d'autres sélecteurs.

  • Définir la mise en forme du sélecteur nav, définir :
    • Une couleur d'arrère plan background-color (code couleur #3f51b5)
    • Une marge extérieure basse margin-bottom à 25 pixels
    • Une marge intérieure padding à 15 pixels
    nav {
    background-color: #3f51b5;
    margin-bottom: 25px;
    padding: 15px;
  }
  • Définir la mise en forme de la balise ul qui est à l'intérieur d'une balise nav, c'est-à-dire nav ul :
    • Aligner la liste non ordonnée au centre de la page, définir la propriété text-align à center
    • Supprimer les bulles de la liste, définir la propriété list-style à none
    • Supprimer les marges intérieures et extérieures, propriétés margin et padding 
    nav ul {
    text-align: center;
    list-style: none;
    padding: 0; margin: 0;
}
  • Mettre toutes les li de la liste précédente sur une même ligne, et définir une marge extérieure horizontale de 20 pixels. 
    nav ul li {
    display: inline-block;
    margin: 0 20px;
}
  • La couleur des liens devra être blanche et leur ôter le soulignement par défaut, qui est propre aux liens hypertextes. Ajouter le soulignement, lorsque le lien est survolé : 
    nav ul li a {
    text-decoration: none;
    color: white;
}
nav ul li a:hover {
  text-decoration: underline;
}

  <div class="carre col-1"></div>

Partie CSS

On commence par changer le style et la taille de la police du body. On enlève toutes les marges. 

body {
  font-family: sans-serif;
  font-size: 40px;
  margin: 0;
  padding: 0;
}

On centre le titre tout en réduisant sa marge extérieure.

h1 {
  text-align: center;
  color: chocolate;
  margin: 20px 0;
}

On définit les dimensions de la grille du jeu width et height. On utilise display: flex pour agencer les carreaux l'un à coté de l'autre. On utilise flex-wrap: wrap pour indiquer que le contenu sera rendu sur plusieurs lignes. Le tout sera indiqué dans #game. 

#game {
  width: 450px;
  height: 450px;
  display: flex;
  flex-wrap: wrap;
  margin: 0 auto;
}

Chaque carreau doit tenir dans une boite de dimensions 150px (450px / 3).

largeur totale = width + 2 × padding + 2 × border

Avec une bordure d'épaisseur égale à 1px et une marge intérieure (padding) de 10px, il reste 128px pour la largeur du box.

Tous les carrés du jeu .carre partagent les mêmes propriétés et différent par leurs couleurs qui sera indiquée dans les classes .col-1 et .col-2.

.carre {
  width: 128px;
  height: 128px;
  padding: 10px;
  font-size: 128px;
  line-height: 128px;
  text-align: center;
  border: #000 solid 1px;
}

Enfin, les carreaux possèdent des couleurs alternées qui sont définis par les classes .col-1 et .col-2. 

.col-1 {
  background-color: lightblue;
}
.col-2 {
  background-color: lightsalmon;
}

Modèle de boite (Box model)

Présentation

Chaque élément HTML est considéré, en CSS, comme étant une boite qui possède :

  • Un contenu (content) : représente une image ou un texte affichés;
  • Des marges intérieures (padding) : un espace transparent entre les bordures de l'élément et son contenu;
  • Une bordure (border)
  • Une marge extérieure (margin) : un espace transparent qui sépare l'élément courant de ses voisins.

Exemple

L'élément div suivant possède :

  • une marge intérieure (padding) de 20px
  • une bordure rouge de 10px
  • une marge extérieure (margin) de 30px

Types de balises

Présentation

En HTML, les balises peuvent être classées selon leur mise en page (layout). Ainsi on distingue deux types de balises :

  • Les balises de bloc (block elements) qui commencent toujours sur une nouvelle ligne et qui s'étendent sur toute la largeur disponible.

    Exemples : p, h1, div, footer, main, aside, etc.

  • Les balises en ligne (inline elements) qui prennent uniquement l'espace qui leur est nécessaire, sans nécessiter obligatarement le retour à la ligne.

    Exemples : b, u, i, strong, img, etc.

Les sélecteurs CSS

Une règle CSS est appliquée pour un sélecteur donné. Ainsi, on distingue une variété de sélecteurs dont les plus importants sont :

  • Sélecteur des éléments (element selector)
  • Sélecteur des classes (classe selector)
  • Sélecteur des identifiant (id selector)

Pour plus de précision, ces sélecteurs peuvent être combinés.

Sélecteur d'éléments

Pour définir la mise en forme de tous les paragraphes on peut écrire :

p {
  color: white;
  background-color: black;
}

Pour définir uniquement la mise en forme des paragraphes qui sont des descendants de main, par exemple, on écrit :

main p {
  color: yellow;
  font-weight: bold;
}

Sélecteur de classes

Supposons que l'on souhaite que certains paragraphes d'un texte soient centrés, et que d'autres soient en gras. Enfin, pourquoi pas ne pas appliquer cette combinaison à d'autres paragraphes.

Dans ce cas, il faudra utiliser des sélecteurs de classes qu'on devra définir selon notre besoin.

.center {
  text-align: center;
}
.bold {
  font-weight: bold;
}

Dans le code HTML, ces styles sont appliqués en utilisant l'attribut class des éléments.

<!-- Centrer le paragraphe -->
<p class="center">...</p>
<!-- Mettre en gras -->
<p class="bold">...</p>
<!-- Centrer le paragraphe et le mettre en gras -->
<p class="center bold">...</p>

Sélecteur d'identifiant

Un élément d'une page HTML peuvent avoir un id qui doit être unique dans une page. Exemple : 

<div id="cercle-bleu">...</div>

Cet élément est ensuite accessible grâce au sélecteur #cercle-bleu en CSS :

#cercle-bleu {
  background-color: blue;
  border-radius: 50%;
}

    body {

Les formulaires

Pour s'inscrire sur un site Web, pour se connecter à Facebook, pour répondre à une enquête, il faut remplir un formulaire.

Un formulaire est utilisé pour recueillir les informations des utilisateurs d'un site Web ou d'une application Web.

Le langage HTML définit les composants du formulaire les plus basiques. Et il existe, aussi, beaucoup de bibliothèques qui offrent des composants personnalisées.

Le formulaire suivant est définit uniquement à l'aide du langage HTML.

Combien de fruits mangez-vous par jour ?

Le code HTML nécessaire pour générer ce formulaire est le suivant :

<form action="destination.php" method="post">
  <div>
    <label for="np">Nom & Prénom</label>
    <input id="np" name="np" type="text" placeholder="Votre nom et prénom">
  </div>
  <div>
    <label for="mp">Mot de passe</label>
    <input id="mp" name="mp" type="password" placeholder="Votre mot de passe">
  </div>
  <div>
    <label for="em">Email</label>
    <input id="em" name="em" type="email" placeholder="Votre email">
  </div>
  <div>
    <label for="age">Age</label>
    <input id="age" name="age" type="age" min="0" max="100" 
           type="number" placeholder="Votre age">
  </div>
  <div>
    <div>Combien de fruits mangez-vous par jour ?</div>
    <label>
      <input type="radio" name="fruit" id="fruits-0" value="0">
      Aucun</label>
    <label>
      <input type="radio" name="fruit" id="fruits-1" value="1">
      Un seul
    </label>
    <label>
      <input type="radio" name="fruit" id="fruits-2" value="2">
      Deux
    </label>
    <label>
      <input type="radio" name="fruit" id="fruits-m" value=">2">
      Plus de deux
    </label>
  </div>
  <div>
    <label for="favori">Votre fruit favori est :</label>
    <select name="favori" id="favori" size="4">
      <option value="Pomme">Pomme</option>
      <option value="Banane">Banane</option>
      <option value="Orange">Orange</option>
      <option value="Poire">Poire</option>
    </select>
  </div>
  <div>
    <label for="qst">Pourquoi vous vous mettez au régime</label>
    <textarea id="qst" name="qst" placeholder="Votre age"></textarea>
  </div>
  <div>
    <button type="submit">S'inscrire</button>
    <input type="reset" value="Effacer">
  </div>
</form>

Le formulaire doit être tapé entre <form>...</form>. Cette balise possède plusieurs attributs importants :

  • action indique la ressource qui se chargera du traitement des données du formulaire.
  • method indique la méthode à utiliser lors de la soumission du formulaire. Les deux méthodes les plus utilisées sont :
    • GET les informations saisies seront ajoutées à l'adresse URL de la ressource mentionnée dans l'attribut action.
    • POST les informations saisies seront envoyées dans le corps de la requête et ne seront pas visibles pour l'utilisateur.

  6. Bouton de soumission, pour soumettre le formulaire.
  7. Bouton de remise à zéro, pour effacer les champs du formulaire.
