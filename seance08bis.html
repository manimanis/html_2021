Les dégradés

Les dégradés linaires

Un dégradé linéaire se propage dans une seule direction.

Exemple

Copier le code HTML suivant :

<div id="grad1" class="box"></div>
<div id="grad2" class="box"></div>
<div id="grad3" class="box"></div>
<div id="grad4" class="box"></div>

Copier le code CSS suivant :

.box {
  width: 720px;
  height: 120px;
  margin: 5px auto;
}

/* Un dégradé équilibré de gauche à droite */
#grad1 { background: linear-gradient(to right, #f00 0%, #00f 100%); }

/* Un dégradé non équilibré de haut en bas */
#grad2 { background: linear-gradient(to bottom, #0f0 0%, #f00 60%); }

/* Un dégradé équilibré d'un angle 70° */
#grad3 { background: linear-gradient(70deg, pink, yellow); }

/* Un dégradé équilibré de trois couleurs d'un angle 70° */
#grad4 { background: linear-gradient(-80deg, red 0%, green 50%, blue 100%); }

Les dégradés radiaux

Un dégradé radial est un dégradé qui va partir d'un point central et se propager dans toutes les directions à partir de ce point.

Exemple

Copier le code HTML suivant :

<div id="grad5" class="box"></div>
<div id="grad6" class="box"></div>
<div id="grad7" class="box"></div>
<div id="grad8" class="box"></div>
<div id="grad9" class="box"></div>

Copier le code CSS suivant :

/* Un dégradé radial du blanc au rouge */
#grad5 { background: radial-gradient(white 0%, red 100%); }

/* Un dégradé radial circulaire de trois couleurs */
#grad6 { background: radial-gradient(circle, yellow, green, purple); }

/* Un dégradé radial de couleurs à positions non équilibrées */
#grad7 { background: radial-gradient(pink 40%, cyan 60%, magenta); }

/* Un dégradé radial avec la position du centre en haut */
#grad8 { background: radial-gradient(at top, #f44336, #ffeb3b, #00bcd4); }

/* Un dégradé radial avec la position du centre définie en pourcentage*/
#grad9 { background: radial-gradient(at 20% 80%, #ffc107, #ff9800, #ff5722); }

Les positionnements

Positionnement statique

Le positionnement statique est celui reçu par défaut par chaque élément. Cela veut tout simplement dire « positionner l'élément selon le flux normal ».

Positionnement relatif

L'élément qui possède un positionnement relatif est positionné par rapport à son emplacement actuel dans la page.

Exemple

Soit le code HTML suivant :

<div class="carre"></div>
<div class="carre1"></div>
<div class="carre"></div>

Et le code CSS correspondant :

.carre,
.carre1 { width: 100px; height: 100px; background-color: red; }

.carre1 {
  background-color: blue;
  position: relative;
  left: 110px;
  top: -100px;
}

Positionnement absolu

L'élément qui possède un positionnement absolu est positionné par rapport à son parent le plus proche qui possède un positionnement relatif.

Exemple

Soit le code HTML suivant :

<div class="coul1">
  <div class="coul2">
    <div class="coul3"></div>
  </div>
</div>

Et le code CSS correspondant :

.coul1, .coul2, .coul3 {
  padding: 20px;
  background-color: red;
}

.coul1 { width: 120px; position: relative; }

.coul2 { background-color: green; }

.coul3 {
  background-color: blue;
  position: absolute;
  top: 0;
  left: 0;
}

Le résultat final est le suivant : le carré bleu .coul3 est supprimé de sa place normale et inséré à la position (0, 0) par rapport au carré rouge .coul1.
