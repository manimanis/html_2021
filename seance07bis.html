Colorful Text

On demande de créer une page Web qui contient le texte suivant :

Travail demandé

  1. Créer un dossier nommé colorful-text, créer dans ce dossier deux fichiers :
    • un fichier HTML nommé colorful-text.html
    • un fichier CSS nommé colorful-text.css
  2. Lier le fichier colorful-text.css à la page Web : <link rel="stylesheet" href="colorful-text.css">
  3. Copier/Coller le code HTML suivant dans la partie <body>...</body> de la page. 
    <div class="container">
    <div class="text">
      <span class="letter color1">C</span><span class="letter color2">O</span><span 
        class="letter color3">L</span><span class="letter color4">O</span><span 
        class="letter color5">R</span><span class="letter color6">F</span><span
        class="letter color7">U</span><span class="letter color8">L</span>
    </div>
    <div class="text">
      <span class="letter color1">T</span><span class="letter color2">E</span><span 
        class="letter color3">X</span><span class="letter color4">T</span>
    </div>
  </div>
  4. Modifier les marges (margin) du sélecteur .container pour centrer son contenu horizontalement.
  5. Modifier les propriétés du sélecteur .text :
    • Utiliser une police Garamond d'une taille de 250px.
    • Mettre en gras (bold) le texte.
    • Centrer le texte de l'élément.
  6. Modifier la largeur du sélecteur .letter à 100px, ne pas oublier de changer l'affichage display de cet élément à inline-block.
  7. Les couleurs des lettres sont comme suit :
    • color1 #214697CC | color2 #198e23CC
    • color3 #f9a60aCC | color4 #70046aCC
    • color5 #de2f1cCC | color6 #e00046CC
    • color7 #1b47aaCC | color8 #f9a60cCC

Carreaux

On demande de :

  • Dessiner les dix carrés précédents de dimensions 150 pixels × 150 pixels chacun.
  • Définir les marges internes (padding) de la façon suivante :
    1. Pas de marges
    2. Marges verticales ⇅ de 20px.
    3. Marges horizontales ⇆ de 20px.
    4. Marges haute et gauche de 20px
    5. Marges droite et basse de 20px
    6. Marges droite et haute de 20px
    7. Marges gauche et basse de 20px
    8. Toutes les marges de 30px
    9. Marges verticlaes de 30px et Marges horizontales de 25px
    10. Marge haute de 50px
  • Mettre les carrés sur deux lignes avec une marge externe (margin) entre deux carrés de 6px.

Code HTML

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <link rel="stylesheet" href="carre.css">
</head>
<body>
  <div class="carre carreA">A</div>
  <div class="carre carreB">B</div>
  <div class="carre carreC">C</div><!-- Compléter les autres carreaux -->
</body>
</html>

Drapeaux à trois bandes

On demande de créer une page Web qui contient des drapeaux à trois bandes :

Travail demandé

  1. Créer un dossier nommé flags, créer dans ce dossier deux fichiers :
    • un fichier HTML nommé flags.html
    • un fichier CSS nommé flags.css
  2. Lier le fichier flags.css à la page Web : <link rel="stylesheet" href="flags.css">
  3. Copier/Coller le code HTML suivant dans la partie <body>...</body> de la page. 
    <div class="container">
  <div id="armenie" class="drapeau horizontaux">
    <div class="bande bande1"></div>
    <div class="bande bande2"></div>
    <div class="bande bande3"></div>
  </div>
</div>
<div class="container">
  <div id="france" class="drapeau verticaux">
    <div class="bande bande1"></div>
    <div class="bande bande2"></div>
    <div class="bande bande3"></div>
  </div>
</div>
  4. Compléter le code HTML manquant pour les trois autres drapeaux de chaque catégorie.
    Copier/Coller le code HTML d'un drapeau tout en changeant juste son id.
  5. Tous les drapeaux :
    • mesurent : 198px × 150px
    • possèdent une marge horizontale (margin left/right) de 20px
    • possèdent une marge inférieure (margin bottom) de 35px
    • possèdent une bordure de 1px de couleur gris
    • utilisent un positionnement relatif
  6. Sous les drapeaux :
    • Afficher le nom du pays en gras et centré sur la totalité de l'espace horizontal disponible.
    • Afficher le nom du pays dans la marge à -25px du bord inférieur du drapeau.
    • Utiliser un positionnement absolu du nom.
  7. Les bandes des drapeaux horizontaux :
    • Mesurent 100% de l'espace horizontal
    • Ont une largeur de 50px
  8. Les bandes des drapeaux verticaux :
    • Mesurent 33% de l'espace horizontal
    • Ont une largeur de 150px
  9. Colorier les bandes des drapeaux selon la figure ci-dessus.
