Les langages du Web

Les principaux langages du web sont :

  • Le langage HTML (HyperText Markup Language) utilisé pour la création d'une page web. Il est constitué de balises qui permettent la structuration de la page et d'autres qui permettent de lier les pages et les ressources entre-elles.
  • Le langage CSS (Cascading Style Sheets) est utile pour la mise en forme d'une page web. L'utilisation de ce langage dans une page la rend plus attractive et plus agréable à naviguer.
  • Le langage JS (JavaScript) ajoute de l'interactivité et améliore l'expérience de l'utilisateur pour des pages plus conviviales et plus ergonomiques.
