Tim Berners-Lee

The man who created the World Wide Web

Tim Berners-Lee is best known as the creator of the world wide web.

In addition to this, he also created the web's foundational languages :

  • HTTP : HyperText Tranfer Protocol
  • HTML : HyperText Markup Language
  • URI : Uniform Resource Identifier

HTML became an introductory language for many programmers and enjoyed a great popularity in the early 2000's.
