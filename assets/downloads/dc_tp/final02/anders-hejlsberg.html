Anders Hejlsberg

The creator of the JavaScript Programming Language

Hejlsberg is a danish programmer best known as the creator of JavaScript, the famous Web programming language that first introduced interactivity to web oages.

He is also the author of :

  • The TypeScript language which is a subset of the JavaScript.
  • The C# language the star language of .Net Framework.
  • The Delphi language an improvement over the Pascal language.

TypeScript is maintained by Microsoft and is compatible with JavaScript.
