Le Secret des Archives Numériques

Codex ouvrit le premier livre. Pour lire les informations, il utilisa d'abord mysqli_fetch_row.

C'était une méthode simple : il lisait les données dans l'ordre où elles apparaissaient, comme des colonnes numérotées.

« Position 0 : Nom, Position 1 : Âge », murmurait-il.

C'était précis, mais il devait se souvenir de l'ordre exact de chaque détail pour ne pas se tromper.
