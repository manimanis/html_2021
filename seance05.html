Les formulaires

Les formulaires HTML sont utilisés par les sites Web pour interagir avec l'utilisateur. Ils permettent à l'utilisateur d'envoyer des données au site web. La plupart du temps, ces données sont envoyées à des serveurs web.

Un formulaire HTML est formé par un ou plusieurs composants. Ceux-ci peuvent être des champs/zones de texte, des boîtes à sélection, des boutons, des cases à cocher ou des boutons radio.

Le formulaire ci-dessus est composé de :

  1. Champ de texte, input, type text, <input type="text" name="nom_prenom">
  2. Champ de texte, input, type password, <input type="password" name="mot_passe">
  3. Champ de texte, input, type email, <input type="email" name="adr_email">
  4. Champ de texte, input, type number, <input type="number" name="age">
  5. Boutons radio, input, type radio, <input type="radio" name="nbr_fruits" value="0">
  6. Liste déroulante, select, avec une ou plusieurs option, 
    <select name="favori">
  <option value="f1">Pomme</option>
  <option value="f2">Banane</option>
  <option value="f3">Orange</option>
</select>
  7. Zone de texte, <textarea name="regime" cols="30" rows="10"></textarea>
  8. Cases à cocher, input, type checkbox, <input type="checkbox" name="contact" id="email" value="0">
  9. Bouton, button, type submit, <button type="submit">S'inscrire</button>
  10. Bouton, button, type reset, <button type="reset">Effacer</button>

Révision

Application 3

Nous construisons le squelette d'un site Web, HTML uniquement, pour le magasin de vente de matériel informatique Techno Store, composé de sept pages :

  • index.html : Page d'accueil qui présente tous les articles en promotion
  • lenovo.html, dell.html, hp.html, msi.html : Caractéristiques des PCs en promotion
  • commander.html : Formulaire de commande d'un produit
  • confirmation.html : Message affiché pour confirmer la commande

Cliquer sur les images pour les agrandir.

Travail demandé

  1. Télécharger le texte et les images nécessaires pour réaliser ce TP : Fichiers du TP
  2. Décompresser le fichier compressé dans votre dossier de travail.
  3. Ouvrir ce dossier à l'aide de Visual Studio Code.
  4. Ajouter la structure minimale d'une page Web à chacune des quatre pages. Ne pas supprimer les commentaires
  5. Répondre en dessous des commentaires sans les supprimer.
