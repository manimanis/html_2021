Les feuilles de style

Introduction

Les feuilles de style en cascade sont définis par le site Mozilla Developper Network (MDN). comme suit :

Les Cascading StyleSheets — ou CSS — (Feuilles de style en cascade) sont la première technique à apprendre après le HTML. Alors que HTML s'utilise pour définir la structure et la sémantique du contenu, les CSS sont employées pour composer et déterminer l'apparence de ce contenu.

Ainsi par exemple, vous utiliserez les CSS pour modifier les polices, la couleur, la taille et l'espacement de votre contenu, pour le répartir sur plusieurs colonnes ou bien pour ajouter des animations et autres fonctionnalités décoratives.

Premiers pas

Pour appliquer un style dans une page Web, on peut opter pour l'une des trois méthodes suivantes :

  • En utilisant l'attribut style : Le style est appliqué directement à l'élément à travers l'attribut style="..." : <p style="...">...</p>.
  • En utilisant une feuille de style interne : En ajoutant, la balise <style>...</style> dans la partie <head>...</head> du document HTML.
  • En utilisant feuille de style externe : La balise link autorise le partage d'une même feuille de style entre plusieurs pages web sans avoir à dupliquer le code plusieurs fois, et ce en utilisant la balise link : <link rel="stylesheet" href="style.css">.

Attribut style

Nous voulons par exemple illustrer les éléments d'une phrase à l'aide de plusieurs couleurs différentes.

  • Le sujet : couleur brown  
  • Les adjectifs : couleurs blue   et red  
  • Le verbe : couleur chartreuse  
  • Le complément : couleur darkgoldenrod  

Le code HTML de cette phrase est comme suit :

<p><span style="color: brown;">Une 
  <i style="color: blue;">jolie</i> 
  <i style="color: red;">petite</i> fille</span> 
  <b style="color: chartreuse;">se promène</b> 
  <u style="color: darkgoldenrod;">dans le bois</u>.</p>

Feuille de style interne

La balise <style>...</style> permet d'intégrer une feuille de style interne dans une page Web.

Normalement, cette balise doit figurer dans la partie <head>...</head> du document.

Nous allons reprdrendre l'exemple précédent qui illustre les composants d'une phrase à l'aide de différentes couleurs.

L'utilisation de cette méthode requiert des changements au niveau du code HTML, dans le but de donner une, certaine, sémantique (sens) à chacune des parties de la phrase.

<p><span class="sujet">Une
    <i class="adjectif1">jolie</i>
    <i class="adjectif2">petite</i> fille</span>
  <b class="verbe">se promène</b>
  <u class="complement">dans le bois</u>.
</p>

Dans la partie <head>...</head>, on définit la feuille de style interne de la façon suivante :

<style>
.sujet {
  color: brown;
}
</style>

La balise <style>...</style> comporte l'ensemble des règles de mise en forme à appliquer au document.

Règles de style

Une règle est appliquée à un sélecteur qui indique les éléments concernés par une règle.

Une règle modifie les valeurs des propriétés du sélecteur.

Dans la figure précédente, .sujet est appelé un sélecteur de classe, qui signifie que la règle sera appliquée à tous les éléments qui appartiennent à la classe sujet, plus précisement toutes les balises définies : <balise class="sujet"></balise>.

La propriété color, qui définit la couleur du texte, de ce sélecteur est définie à brown.

Feuille de style externe

Plus fréquemment, la feuille de style est placée à l'extérieur du document html, dans un fichier qui porte l'extension .css,exemple : style.css.

Cette pratique est très recommandée dans la conception d'un site web car :

  • Elle favorise le partage d'un même style pour toutes les pages pour obtenir une apparence homogène.
  • Elle évite la duplication du code CSS dans plusieurs documents et donc facilite la maintenance et les modifications.

Une feuille de style est liée à une page web à travers la balise <link> : <link rel="stylesheet" href="style.css">

On adoptera cette méthode dans le reste de ce cours.

Alignement du texte

Nous allons commencer par centrer le texte de la page. La propriété text-align permet de définir l'alignement du texte :

h1 {
  text-align: center;
}

p {
  text-align: center;
}

Coloriage du texte

La propriété color définit la couleur du texte d'un élément.

h1 {
  /* ajouter cette propriété */
  color: #e30000;
}

p {
  /* ajouter cette propriété */
  color: #0081a3;
}

Taille de la police

La propriété font-size définit la taille du texte.

h1 {
  /* ajouter cette propriété */
  font-size: 42pt;
}

p {
  /* ajouter cette propriété */
  font-size: 42pt;
}

Famille de la police

La propriété font-family définit la famille et le style d'écriture du texte.

Le sélecteur @font-face est ajouté lorsque les polices utilisées sont non disponibles sur la machine de l'utilisateur.

@font-face {
  font-family: Dancing Script;
  src: url(fonts/DancingScript-VariableFont_wght.ttf);
}

@font-face {
  font-family: Lily Script;
  src: url(fonts/LilyScriptOne-Regular.ttf);
}

h1 {
  /* ajouter cette propriété */
  font-family: 'Lily Script', cursive;
}

p {
  /* ajouter cette propriété */
  font-family: 'Dancing Script', cursive;
}

Cacher le texte d'un élément

Pour cacher le texte à l'intérieur du span il faudra agir sur trois propriétés.

  • display : par défaut la balise span est une balise inline, c'est à dire qu'on ne peut pas agir sur ces dimensions. Nous allons modifier la valeur cette propriétés à inline-block, c'est à dire un élément inline dont on peut ajuster les dimensions width et height.
  • width : la largeur par de l'élément span
