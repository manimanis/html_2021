Problématique

Le QCM suivant, composé de 6 questions, est destiné aux élèves de la maternelle : Cliquer pour afficher le formulaire complet.

Les attributs du formulaire sont comme suit :

  • La méthode HTTP : POST
  • L'action : Demander à votre enseignant

Question 1

La première question ressemble à ce qui suit :

Cette question utilise deux boutons radios : <input type="radio">

  • L'attribut name doit être le même pour les deux boutons: name="genre"
  • L'attribut value doit être : Garçon pour le premier bouton et Fille pour le second.
  • L'attribut required doit être utilisé pour l'un des deux boutons.

Les deux boutons radio sont dans un <div></div>. Et, chaque bouton radio doit faire partie d'un <label></label>.

Image à télécharger pour l'utiliser dans la question :

  <div>
    <label><input type="radio" name="genre" value="Garçon" required> Un garçon</label>
    <label><input type="radio" name="genre" value="Fille"> Une fille</label>
  </div>
</div>

Question 4

La quatrième question ressemble à ce qui suit :

Dans cette question on utilise deux champs de texte :

  1. Le premier est de type texte : <input type="text">
    • Les attributs name et id sont tous les deux identiques nom_fille : name="nom_fille"
    • L'attribut required doit être utilisé avec ce champ.
  2. Le second est de type number : <input type="number">
    • Les attributs name et id sont tous les deux identiques age_fille : name="age_fille"
    • Les attributs min et max son utilisés pour fixer l'intervalle d'âge valide à [1, 10].
    • L'attribut required doit être utilisé avec ce champ.

Question 5

La cinquième question ressemble à ce qui suit :

Dans cette question on utilise une zone de texte .

La zone de texte texte <textarea></textarea> possède les attributs suivants :

  • Les attributs name et id sont tous les deux identiques phrases : name="phrases"
  • Les attributs cols et rows sont utilisés pour fixer le nombre de caractères affichés horizontalement et le nombre de lignes affichées verticalement. Dans notre cas ces valeurs sont comme suit :
    • Nombre de colonnes = 60, cols="60"
    • Nombre de lignes = 30, rows="3"
  • L'attribut required doit être utilisé avec ce champ.

Soumission du formulaire

Prévoir un <div></div> contenant deux boutons :

  • Bouton d'envoi <button type="submit">Vérifier</button>
  • Bouton de remise à zéro type="reset" qui permet d'effacer les champs du formulaire.

Tester votre formulaire après avoir complété l'attribut action="..." de la balise <form action="..."></form>. Cette information sera indiquée par votre enseignant.
