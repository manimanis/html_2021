Tic-Tac-Toe Game

On veut concevoir la plateforme du jeu, très connu : Tic-Tac-Toe

Travail Demandé

  1. Créer un dossier nommé : tictactoe. Créer dans ce dossier deux fichiers :
    • tictactoe.html
    • tictactoe.css
  2. Lier la feuille de style tictactoe.css au fichier HTML tictactoe.html.
  3. Copier/Coller le code HTML suivant dans la partie <body>...</body> de la page. 
    <h1>Tic-Tac-Toe Game</h1>
<div id="game">
  <div class="carre col-1">X</div>
  <div class="carre col-2">O</div>
  <div class="carre col-1">X</div>
  <div class="carre col-2">O</div>
  <div class="carre col-1"></div>
  <div class="carre col-2">X</div>
  <div class="carre col-1"></div>
  <div class="carre col-2"></div>
  <div class="carre col-1"></div>
</div>
  4. Mettre en forme le titre <h1>...</h1>:
    • Police : sans-serif, Taille : 40px
    • Titre centré avec une marge verticale de 40px
    • Couleur : #007096
  5. Mettre en forme l'élément ayant game comme id :
    • Largeur : 450 pixels
    • Display : flex avec retour à la ligne autmoatique
    • Marges horizontales : automatiques
    • Bordure de 15 pixels de couleur #666
  6. Mettre en forme l'élément ayant carre comme classe :
    • Dimensions : 120 pixels × 120 pixels
    • Marge intérieure : 9 pixels
    • Marge extérieure : 3 pixels
    • Texte centré, Taille : 120 pixels, Style : Gras
    • Bordure de 3 pixels couleur #000
    • Hauteur de lignes (line-height) : 120 pixels
  7. Mettre en form les éléments ayant col-1 et col-2 comme classes :
    • Couleur d'arrière plan de col-1 : lightblue
    • Couleur d'arrière plan de col-2 : lightsalmon

Correction - Tic-Tac-Toe Game

Contenu du fichier : tictactoe.css

body {
  font-family: sans-serif;
  font-size: 20px;
  padding: 0;
}

h1 {
  text-align: center;
  color: #007096;
  margin: 40px 0;
}

#game {
  width: 450px;
  display: flex;
  flex-wrap: wrap;
  margin: 0 auto;
  border: #666 solid 15px;
}

.carre {
	width: 120px;
	height: 120px;
	padding: 9px;
	margin: 3px;
	font-size: 120px;
  font-weight: bold;
	line-height: 120px;
	text-align: center;
	border: #000 solid 3px;
}

.col-1 {
  background-color: lightblue;
}

.col-2 {
  background-color: lightsalmon;
}
